# Networking Fundamentals Notes

A beginner-friendly networking reference covering IP addressing, DNS, HTTP/HTTPS, ports, protocols, VPNs, and basic troubleshooting.

This project is designed as a practical study resource for learners entering IT, cybersecurity, networking, or software engineering roles.

## What this project covers

- IP addressing and subnetting basics
- DNS and how domain names resolve
- HTTP vs HTTPS
- Common ports and protocols
- VPN fundamentals
- Basic network troubleshooting commands
- Simple networking diagrams using Mermaid

## Who this is for

- Beginners learning networking
- Students preparing for IT or cybersecurity roles
- Software engineering learners who want to understand how web communication works
- Anyone building foundational technical knowledge

## How to use this repo

Start with the files inside the `notes/` folder in order. Then use the troubleshooting and command cheat sheets for revision or interview preparation.

## Key learning outcomes

After going through this project, you should be able to:

- Explain how devices communicate over a network
- Understand the purpose of IP addresses, ports, and protocols
- Describe what happens when a website loads in a browser
- Use basic commands to troubleshoot connectivity issues
- Identify commonly used network services and ports
